Tree Traversal

1. Inorder Traversal
    - All nodes in left subtree
    - Root Node
    - All nodes in right subtree
2. Preorder Traversal
    - Root Node
    - All nodes in left subtree
    - All nodes in right subtree
3. Postorder Traversal
    - All nodes in left subtree
    - All nodes in right subtree
    - Root Node
